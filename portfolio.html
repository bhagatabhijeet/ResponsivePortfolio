MeteoPal is an application built on top of openweathermap API. The application is both web browser and mobile browser friendly. The application allows users to search any city in the open weathermap data base. The application fetches the city's current info, forecast for 5 days and shows the UVIndex (color coded e.g Red means harmful, Violet means Extreme and Green means safe) of the city.
